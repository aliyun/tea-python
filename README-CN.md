## 重要的更新

- `darabonba-core` 包版本从 1.0.0 开始，仅支持 Python 3.7 及以上的环境。

## 安装

- **使用 pip 安装(推荐)**

如未安装 `pip`, 请先至pip官网 [pip user guide](https://pip.pypa.io/en/stable/installing/ "pip User Guide") 安装pip .

```bash
# 安装 darabonba-core
pip install darabonba-core
```
